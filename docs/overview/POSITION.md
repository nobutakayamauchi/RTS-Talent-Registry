# POSITION: External AI Talent Governance Registry (Pre-Test Skeleton)

## Intent
This repository provides a minimal governance ledger for evaluating external AI talent candidates in the RTS ecosystem.

## Non-Goals
- Not an authority source for external artifacts.
- Not a deployment or runtime execution environment.
- Not a storage location for upstream manifest bodies.

## Operating model
- Candidate-first intake
- Evidence-based scoring
- Controlled promotion stages
- Route assignment by role fit and risk
- Retirement when stale/high-risk/unsupported

## Classification model
Use the following evidence partitions in records:
- `confirmed_facts`
- `assumptions`
- `unverified`
- `risks`
